¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Es como el esqueleto de una página web, definiendo su estructura y contenido. 


<!DOCTYPE html>
<html>
    <head>
        <title>Mi Primera Página</title>
    </head>
    <body>
        <h1>¡Hola Mundo!</h1>
    </body>
</html>

¡Hola Mundo!

Etiquetas HTML Esenciales

Encabezados <h1> - <h6>

Los encabezados definen la jerarquía y estructura del contenido.


<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>

Encabezado 1

Encabezado 2

Encabezado 3

Párrafos <p>

Los párrafos son bloques de texto que ayudan a organizar el contenido.


<p>Este es un párrafo de ejemplo.
   Puede contener múltiples líneas
   de texto.</p>

Este es un párrafo de ejemplo. Puede contener múltiples líneas de texto.

Imágenes <img>

Las imágenes enriquecen el contenido visual de la página.


<img src="imagen.jpg" 
     alt="Descripción de la imagen">
Enlaces <a>

Los enlaces permiten la navegación entre páginas web.


<a href="https://ejemplo.com">
   Visitar sitio</a>

Más Etiquetas HTML

Listas <ul> <ol> <li>

Crea listas ordenadas y no ordenadas.


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
<ol>
    <li>Primero</li>
    <li>Segundo</li>
</ol>
  • Item 1
  • Item 2
  1. Primero
  2. Segundo
Tablas <table>

Organiza datos en filas y columnas.


<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>
Encabezado 1 Encabezado 2
Dato 1 Dato 2
Formularios <form>

Crea formularios interactivos.


<form>
    <label>Nombre:</label>
    <input type="text">
    <label>Email:</label>
    <input type="email">
    <button type="submit">
        Enviar
    </button>
</form>
Multimedia <video> <audio>

Incorpora contenido multimedia.


<video controls>
    <source src="video.mp4" 
            type="video/mp4">
</video>

<audio controls>
    <source src="audio.mp3" 
            type="audio/mpeg">
